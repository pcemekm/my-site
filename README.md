# My Site - Personal Website

Personal website built with React, Vite, and Chakra UI.

## Quick Start

### Development

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

### Production Build

```bash
npm run build
npm run preview
```

### Docker

```bash
# Build and run
docker-compose up -d

# View logs
docker-compose logs -f

# Stop
docker-compose down
```

## Access

- **Development**: http://localhost:5173
- **Production**: http://pcemek.local
- **Docker**: http://localhost:3010

## CI/CD

Automatically deployed via Jenkins on every push to the repository.
